    # HUE-BIT-OS

Just a simple file manager where you can drop files and manage stuff.

## What it does

- Drop any file (images, videos, audio, whatever) into the drop zone
- Has a volume slider for audio playback
- User profile stuff with some controls
- Lock, refresh, and power buttons

## How to use it

1. Just open the `index.html` file in your browser
2. Drag and drop files where it says "Drop files to import"
3. Adjust volume with the slider if needed
4. That's pretty much it

## Controls

- 🔊 Volume slider - adjust audio volume
- 👤 User - access your profile
- 🔒 Lock - lock your session
- 🔄 Refresh - reload stuff
- ⏻ Power - power controls

## Tech stuff

Built with HTML, CSS, and JavaScript. Nothing fancy.
